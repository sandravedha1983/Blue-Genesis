# Blue Genesis – The Living Ocean Interface

Blue Genesis is a futuristic, AI-integrated platform designed to showcase the potential of the Blue Economy. It combines a cinematic frontend with a robust FastAPI backend to provide real-time marine intelligence, automated research status, and AI-powered ocean simulations.

## 🌊 Key Features

### 1. Intelligent Ecosystem Dashboard
Explore three distinct marine zones:
- **AquaVeda (Surface)**: Sustainable seaweed farming and biomass production.
- **OceanInk (Mesopelagic)**: Advanced biopolymer and biomaterial research.
- **BioVaultX (Abyssal)**: Deep-ocean genetic discovery and conservation.

### 2. AI Visual Systems
- **Dynamic Backgrounds**: High-fidelity ocean environments generated by Gemini AI.
- **Cinematic Experience**: Immersive video backgrounds and glass-morphism UI components.
- **Nereus AI Assistant**: An interactive global assistant providing status-driven ocean insights.

### 3. Real-time Backend Integration
- **Centralized API Architecture**: All frontend calls are routed through a robust `fetchAPI` helper for consistency and reliability.
- **Live Status Feed**: Continuous updates from marine farms and research facilities.
- **Ocean Simulator**: Input environmental parameters to predict ocean zones via the machine learning backend.
- **Research Vault**: Dynamic database of marine findings and species discoveries.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6).
- **Backend**: FastAPI (Python), Uvicorn.
- **AI**: Gemini AI (for image generation and assistant logic).
- **Architecture**: Centralized API helper with async/await integration logic.

## 🚀 Getting Started

### Prerequisites
- Python 3.9+
- FastAPI & Uvicorn (`pip install fastapi uvicorn`)

### Installation & Run
1. Clone the repository.
2. Start the **Backend** (Server):
   ```bash
   python main.py
   ```
3. Start the **Frontend** (Interface):
   ```bash
   python serve.py
   ```
4. Access the platform at: **[http://127.0.0.1:5500](http://127.0.0.1:5500)**

## 🔗 Backend API Endpoints
- `GET /farms/status` – Live farming status.
- `GET /products/status` – Biomaterial production updates.
- `GET /research/status` – Latest genetic research findings.
- `POST /ocean/predict-zone` – AI model for ocean zone mapping.
- `GET /auth/status` – Nereus AI health check and core auth status.
